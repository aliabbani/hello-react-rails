# Hello React-Rails
The project is about learning how to set up applications using the react-rails gem on Ruby on Rails.
## Built With
### Back-end
- Ruby 
- Ruby on Rails

### Front-end
- React
- Redux
## Getting Started
To get a local copy up and running follow these simple example steps.

### Prerequisites
Ruby: 2.6.3 Rails: 5.2.3 Postgres: >=9.5

### Setup
Install gems with:
```javascript
bundle install
```
Setup database with:
```javascript
rails db:create
rails db:migrate
rails db:seed
```
### Usage
Start server with:
```javascript
rails server
```
Open http://localhost:3000/ in your browser.
